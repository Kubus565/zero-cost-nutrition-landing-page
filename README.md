# Landing Page Sprzedażowy | IgaDietetyk

Profesjonalna, wysokowydajna strona sprzedażowa zaprojektowana dla dietetyka w celu automatyzacji pozyskiwania klientów.

🔗 **Adres strony:** [igadietetyk.pages.dev](https://igadietetyk.pages.dev)

## 🎯 Cel Biznesowy
Strona jest **narzędziem sprzedażowym** skoncentrowanym na konwersji (umówienie płatnej konsultacji). 

## 💡 Rozwiązane problemy
- **Koszt utrzymania 0 PLN:** Wykorzystanie darmowego hostingu i domeny .pages.dev w **Cloudflare**.
- **Wydajność:** Dzięki generatorowi Hugo strona serwuje statyczny HTML, co skraca czas ładowania do minimum.
- **Automatyzacja (CI/CD):** Każda zmiana w kodzie na GitHubie automatycznie aktualizuje stronę na produkcji.

## 🛠️ Stack Techniczny
- **Generator:** Hugo (Go-based Static Site Generator)
- **Frontend:** HTML5, CSS3
- **Deployment:** GitHub + Cloudflare Pages (Continuous Deployment)
